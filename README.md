# BUG-Reporter

这是一个基于Worktile API开发的iOS应用BUG提交SDK

使用此SDK，测试人员可以快捷地在手机上提交BUG至worktile中，每一个BUG的提交过程不超过1分钟。

## BUG提交过程演示

* 手指在任何时候，于手机屏幕右侧向左划动，即可调起BUG提交界面。
